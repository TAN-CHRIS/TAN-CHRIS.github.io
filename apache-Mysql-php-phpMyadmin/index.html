启用 apache

在MacOS中，apache已经被预先安装好了，所以我们只需要启动它就好了。

在终端中，使用 sudo apachectl start 命令打开apache服务

启动后，打开浏览器输入localhost,界面显示 It Works! 表示apache启用成功。
