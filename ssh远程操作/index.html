基本操作

  1. cd

    • cd / : 进入系统根目录

    • cd.. 或 cd..//

  2. ls 显示当前目录下的文件

  3. mkdir folder: 创建目录，若目录存在会报错

    rmdir folder: 删除空文件

    rm -rf folder: 删除非空文件夹

  4. rm file : 删除非空文件

    rm -r floder : 删除非空目录

  5. cat file: 显示file这个文件的内容
